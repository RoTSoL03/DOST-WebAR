# DOST WebAR Mascot Experience 

## **Goal**

Develop a browser-based WebAR application that allows event attendees to view and take photos with DOST mascots using their mobile devices without installing an application.

## **Target Users**

* Conference attendees  
* Students  
* Researchers  
* Government officials  
* General public

## **Platforms**

* Android Chrome  
* iPhone Safari  
* Mobile browsers only

## **Supported Devices**

* AR-capable Android phones  
* Modern iPhones  
* No desktop support required

## **Core Features**

* QR code launch  
* Camera permissions  
* Plane detection  
* Choose among four mascots  
* Place mascot  
* Rotate  
* Scale  
* Screenshot  
* Download photo

## **Non-functional Requirements**

* Load within 5 seconds  
* 30 FPS minimum  
* Responsive UI  
* Offline asset caching  
* Secure HTTPS deployment
